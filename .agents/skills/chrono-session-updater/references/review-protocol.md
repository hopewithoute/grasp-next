# Human Review Protocol

For every proposed memory update, ask the human to classify each item:

- `accept` — add to accepted chronicle/decision vault
- `reject` — discard
- `edit` — revise wording before storing
- `needs evidence` — keep as open question until supported
- `move to ADR` — promote to architecture-level decision
- `keep feature-level` — preserve only in this feature memory

Never silently accept a new decision just because code was generated.
